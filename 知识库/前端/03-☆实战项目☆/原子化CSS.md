# 原子化CSS

原子化CSS是一种CSS架构方式，其支持小型、单一用途的类，其名称基于视觉功能。

```
<!-- 原子化类定义 -->
<style>
  .text-white { color: white; }
  .bg-black { background-color: black; }
  .text-center { text-align: center; }
</style>

<!-- 原子化类使用 -->
<div class="text-white bg-black text-center">hello Atomic CSS</div>
```
