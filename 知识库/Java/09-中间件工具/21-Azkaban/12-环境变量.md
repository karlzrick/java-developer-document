# 环境变量

### 1、配置文件准备

[env.project](./12-job/env.project)

```yaml
azkaban-flow-version: 2.0
```

[env.flow](./12-job/env.flow)

```yaml
config:
  name: zq

nodes:
  - name: JobA
    type: command
    config:
      # 自定义环境变量
      env.branch: dev
      # 读取环境变量
      command: echo "branch:${env.branch} -- do JobA -- name:${name}"
```
