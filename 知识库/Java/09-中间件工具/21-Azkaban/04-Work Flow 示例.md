#### 1、配置文件准备

[first.project](04-job/first.project)

文件作用：采用新的`Flow-API`方式解析flow文件

```
azkaban-flow-version: 2.0
```

[first.flow](04-job/first.flow)

```yaml
nodes:
  - name: jobA # job名称
    type: command # job类型
    config:
      command: echo "Hello World" # 执行命令
```
