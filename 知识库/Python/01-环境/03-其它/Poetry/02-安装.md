# 安装

```shell
curl -sSL https://install.python-poetry.org | python3 -
```

安装完成后，可以通过 `poetry --version` 来验证安装是否成功。

### 常用命令

- **初始化项目**：`poetry init`
- **添加依赖**：`poetry add <package>`
- **移除依赖**：`poetry remove <package>`
- **安装依赖**：`poetry install`
- **构建包**：`poetry build`
- **发布包**：`poetry publish`
